## Background

For companies on free tier or without a provisioned number, the platform needs a shared fallback number.

---

## Implementation

1. Add environment variable: `PLATFORM_ESCALATION_NUMBER`
2. Update T-015's fallback logic to use it
3. Configure the number in Twilio with generic greeting
